Install

Tip: When downloading, using Google Chrome seems to be the best way to download the game without any problems.

Installation Steps

  1. Download the Eventide XI launcher from the official website.
  2. Extract the downloaded ZIP file to a folder of your choice.
  3. Run Install_Eventide.bat to install the necessary components.

Once everything has been downloaded and all the files have been unzipped, the game should run once you run Launch_Eventide.bat (highlighted in yellow).
