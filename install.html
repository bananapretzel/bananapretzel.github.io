How to Play

Installing with the Launcher

  1. Press the play now button to download EventideLauncher-win-Setup.exe.
  2. Run the eventideLauncher.exe and install.
  3. Once the launcher has been installed, open it up and click the Download button to begin your download. Follow the prompts from the launcher.
  4. Once the installation is done, you can go to the Extensions and Settings pages to modify your addons, plugins, and other configurations.
  5. If you do not have an Eventide game account yet, you can register for one or put in your login information and click PLAY.

FAQs

Is the launcher using Ashita only?

Yes, the Eventide XI launcher is built on Ashita 4. Ashita is a popular third-party FFXI client that provides various enhancements and features for players.


Is it safe to use the launcher?

Yes, using the Eventide XI launcher is safe. The server has implemented security measures to protect players' accounts and data. However, always ensure you download the launcher from the official Eventide XI website to avoid any potential security risks.


How much space do I need for the launcher and the game?

Allot approximately 185MB for the launcher, and 25GB for the game.


Can I use my existing FFXI installation with the launcher?

No, the Eventide XI launcher uses it's own FFXI files. You cannot use your existing FFXI installation with the launcher.


Where can I get support if I encounter issues with the launcher?

If you encounter any issues with the Eventide XI launcher, you can seek support through the official Eventide XI Discord server. The community and staff members are usually helpful in resolving any problems you may face.


Can I customize the launcher settings?

Yes, the Eventide XI launcher allows you to customize various settings, including graphics options, keybindings, and addon configurations. You can access these settings through the launcher's interface.


Does the launcher store my password in plain text?

No and yes. Your information is stored in a maximum of two places: the ashita.ini, and the launcher user.json files. By default, if you put in your username and password in the launcher and click PLAY, your ashita.ini file stores your information in the file in plain text. This is true for any server. The launcher stores your information in its own user.json storage file if you check the "Remember Credentials" box, however your password is encrypted in the user.json file. The best way around storing your password in plain text is to not input your username/password on the login page and manually enter it each time to play.


Is the launcher compatible with all operating systems?

The Eventide XI launcher is primarily designed for Windows operating systems. While it may work on other platforms using compatibility layers or virtual machines, official support is focused on Windows.
